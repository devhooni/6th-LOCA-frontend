# Project Overview

## Problem

기존 지도 서비스는 장소 검색에는 강하지만, 개인이 경험한 장소와 감정, 방문 기록을 나만의 방식으로 관리하기에는 한계가 있습니다.

## Target User

- 홍대 주변을 자주 탐색하는 사용자
- 카페, 음식점, 산책 장소, 포토 스팟을 기록하고 싶은 사용자
- 지도에 없는 개인적인 장소를 저장하고 공유하고 싶은 사용자

## MVP Scope

- Home
- Explore
- For You
- Map
- Place Detail
- Review Write
- My Page
- Admin
- Public Place / Private Place 구분
- Mock fallback 기반 화면 확인

## Out of Scope for This Week

- 신규 API 연동
- For You 고도화 추천
- 최종 디자인 확정
- 사용자 인증 완성
- 대규모 기능 추가

## Product Direction

LOCA는 단순히 장소를 찾는 서비스가 아니라, 장소와 경험을 기록하고 나만의 컬렉션으로 공유하는 서비스입니다.
